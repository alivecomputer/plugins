## Alive Computer

**Your context is your property.**

We build infrastructure for people who use AI to get real work done — and want to own what they build, not rent it.

### The Plugin

**[alive](https://github.com/alivecomputer/claude-plugins)** — Personal private context infrastructure for Claude Code. 12 skills, 12 hooks, a caretaker runtime. Plain markdown files on your machine that any AI agent can read, work in, and hand off.

```bash
claude plugin install alive@alivecomputer
```

### The Thesis

AI makes you productive. But productivity without memory is a treadmill. Every session starts from zero. Every agent forgets what the last one learned. Your context — decisions, relationships, domain knowledge, work history — scatters across chat logs nobody will ever read again.

ALIVE treats context as property. Structured files you own. A runtime any model can inhabit. Zero-context handoff between sessions, agents, and platforms. Nothing phones home. Nothing leaves without your say.
